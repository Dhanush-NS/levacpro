1. History of Python

Python was created by Guido van Rossum and first released in 1991. It was designed to be easy to read and write, making it a great choice for beginners.

Key features include:

  • Readability: Python's syntax emphasizes readability, allowing developers to express concepts in fewer lines of code.
  • Interpreted Language: Python code is executed line by line, which makes debugging easier.

Python has grown significantly since its inception, becoming one of the most popular programming languages worldwide.

2. Key Features of Python

  • Simplicity: Python's syntax is clean and straightforward, making it an excellent choice for beginners.
  • Versatility: Python can be used for web development, data analysis, artificial intelligence, scientific computing, and more.
  • Rich Libraries: Python has a vast standard library and a large ecosystem of third-party packages.
  • Community Support: Python has a large and active community, providing extensive documentation and support.
  • Cross-Platform: Python runs on various platforms, including Windows, macOS, and Linux.

3. Getting Started with Python

3.1 Setting Up Your Environment

To get started with Python, you need to install Python and set up an IDE. Follow these steps:

  • Download Python: Visit the Python download page and install the appropriate version for your system.
  • Set Up an IDE: Use IDEs like PyCharm, Visual Studio Code, or Jupyter Notebook for an enhanced coding experience.

3.2 Writing Your First Python Program

Once your environment is set up, create a new Python file and write your first program:

print("Hello, World!")

3.3 Running Your Program

To run your program, use the command line or your IDE's run feature:

python hello.py

4. Variables and Data Types in Python

In Python, variables are used to store data, and each variable can hold different data types. Python supports various data types, including:

4.1 Basic Data Types

  • int: Stores integers, e.g., age = 30
  • float: Stores floating-point numbers, e.g., salary = 85000.50
  • str: Stores strings, e.g., name = "John"
  • bool: Stores True or False values, e.g., is_python_fun = True

5. Python Operators

Operators in Python perform various operations on variables and values. Here are the types of operators:

5.1 Arithmetic Operators

Used for basic arithmetic operations:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ** (Exponentiation)
  • // (Floor Division)
x = 5
y = 2
print(x + y)  # Outputs: 7
print(x % y)  # Outputs: 1

5.2 Comparison Operators

Used to compare values:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
a = 10
b = 20
print(a == b)  # Outputs: False
print(a < b)  # Outputs: True

5.3 Logical Operators

Used to perform logical operations:

  • and (Logical AND)
  • or (Logical OR)
  • not (Logical NOT)
is_adult = True
has_id = False
print(is_adult and has_id)  # Outputs: False
print(is_adult or has_id)   # Outputs: True
print(not is_adult)         # Outputs: False

5.4 Assignment Operators

Used to assign values to variables:

  • = (Assigns a value)
  • += (Adds and assigns)
  • -= (Subtracts and assigns)
  • *= (Multiplies and assigns)
  • /= (Divides and assigns)
  • %= (Modulus and assigns)
a = 5
a += 3  # a = a + 3, now a is 8
a *= 2  # a = a * 2, now a is 16

6. Conditional Statements in Python

Python supports several types of conditional statements that control the flow of execution based on certain conditions.

6.1 if-else Statement

age = 18
if age >= 18:
print("You are an adult.")
else:
print("You are not an adult.")

6.2 elif Statement

The elif statement can be used to check multiple expressions:

day = 3
if day == 1:
print("Monday")
elif day == 2:
print("Tuesday")
elif day == 3:
print("Wednesday")
else:
print("Unknown day")

7. Loops in Python

Loops are used to execute a block of code repeatedly. Python supports for and while loops.

7.1 for Loop

A for loop is commonly used for iterating over a sequence (like a list or range):

for i in range(5):
print(i)  # Outputs: 0, 1, 2, 3, 4

7.2 while Loop

The while loop continues to run as long as a condition is true:

i = 0
while i < 5:
print(i)
i += 1

8. Functions in Python

Functions are reusable blocks of code designed to perform specific tasks. Here's how to create and call functions in Python:

8.1 Basic Function Example

def add(a, b):
return a + b

result = add(10, 20)
print("Result:", result)  # Outputs: 30

8.2 Function with Default Parameters

You can define functions with default parameter values:

def greet(name="World"):
print("Hello,", name)

greet()          # Outputs: Hello, World
greet("Alice")  # Outputs: Hello, Alice
