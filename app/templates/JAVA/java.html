1. History of Java

Java was developed by Sun Microsystems in the mid-1990s and officially released in 1995. The language was designed to be portable, object-oriented, and user-friendly.

Key features include:

  • Write Once, Run Anywhere (WORA): Java can run on any platform with a Java Virtual Machine (JVM).
  • Object-Oriented: Java promotes code organization and reusability through objects.

In 2010, Oracle acquired Sun Microsystems and has continued to evolve Java with regular updates.

2. Key Features of Java

  • Simplicity: Java's syntax is clear and easy to understand for beginners.
  • Object-Oriented: Promotes better code organization using concepts like encapsulation, inheritance, and polymorphism.
  • Platform Independence: Java programs can run on any platform with a JVM.
  • Rich API: Java includes a wide range of libraries and APIs.
  • Strong Typing: Java helps prevent errors by requiring specific data types.

3. Getting Started with Java

3.1 Setting Up Your Environment

To get started with Java, you need to install the Java Development Kit (JDK) and set up an IDE. Follow these steps:

  • Download the JDK: Visit the Oracle JDK download page and install the appropriate version for your system.
  • Set Up an IDE: Use IDEs like Eclipse, IntelliJ IDEA, or NetBeans for an enhanced coding experience.

3.2 Writing Your First Java Program

Once your environment is set up, create a new Java project and write your first program:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

3.3 Compiling and Running Your Program

  • Compile: In your IDE, use the "Run" or "Compile" button, or use the command line:
    • javac HelloWorld.java
  • Run: Once compiled, run the program with:
    • java HelloWorld

6. Variables and Data Types in Java

In Java, variables are used to store data, and each variable has a specific data type. Java supports various primitive and reference data types.

6.1 Primitive Data Types

  • int: Stores integers without decimal points, e.g., int age = 30;
  • double: Stores floating-point numbers, e.g., double salary = 85000.50;
  • char: Stores single characters, e.g., char grade = 'A';
  • boolean: Stores true or false values, e.g., boolean isJavaFun = true;

6.2 Reference Data Types

Reference types store addresses to objects rather than values directly. Arrays and objects are examples of reference types:

String name = "John"; // Reference type example
int[] numbers = {1, 2, 3, 4}; // Array as a reference type

7. Java Operators

Operators in Java perform various operations on variables and values. Here's a breakdown of all the types of operators:

7.1 Arithmetic Operators

Used for performing basic arithmetic operations.

  • + (Addition) - Adds two values
  • - (Subtraction) - Subtracts one value from another
  • * (Multiplication) - Multiplies two values
  • / (Division) - Divides one value by another
  • % (Modulus) - Returns the remainder of a division
  • ++ (Increment) - Increases a value by 1
  • -- (Decrement) - Decreases a value by 1
int x = 5;
int y = 2;
System.out.println(x + y); // 7
System.out.println(x % y); // 1
System.out.println(++x);   // 6 (Increment)

7.2 Comparison Operators

Comparison operators are used to compare two values and return a boolean result (true/false).

  • == (Equal to) - Checks if two values are equal
  • != (Not equal to) - Checks if two values are not equal
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
int a = 10;
int b = 20;
System.out.println(a == b);  // false
System.out.println(a < b);   // true

7.3 Logical Operators

Used to perform logical "AND", "OR", and "NOT" operations.

  • && (Logical AND) - Returns true if both conditions are true
  • || (Logical OR) - Returns true if at least one condition is true
  • ! (Logical NOT) - Reverses the result of a condition
boolean isAdult = true;
boolean hasID = false;
System.out.println(isAdult && hasID);  // false
System.out.println(isAdult || hasID);  // true
System.out.println(!isAdult);          // false

7.4 Assignment Operators

Used to assign values to variables.

  • = (Assigns a value to a variable)
  • += (Adds and assigns the value)
  • -= (Subtracts and assigns the value)
  • *= (Multiplies and assigns the value)
  • /= (Divides and assigns the value)
  • %= (Takes modulus and assigns the value)
int a = 5;
a += 3;  // a = a + 3, now a is 8
a *= 2;  // a = a * 2, now a is 16

7.5 Bitwise Operators

Operate on bits and perform bit-by-bit operations.

  • & (Bitwise AND)
  • | (Bitwise OR)
  • ^ (Bitwise XOR)
  • ~ (Bitwise NOT)
  • << (Left shift)
  • >> (Right shift)
int a = 5;  // 0101 in binary
int b = 7;  // 0111 in binary
System.out.println(a & b);  // 0101 (5 in decimal)
System.out.println(a | b);  // 0111 (7 in decimal)

7.6 Ternary Operator

The ternary operator is a shortcut for if-else statements.

Syntax: condition ? value_if_true : value_if_false

int age = 18;
String result = (age >= 18) ? "Adult" : "Minor";
System.out.println(result);  // Outputs: Adult

8. Conditional Statements in Java

Java supports several types of conditional statements that control the flow of execution based on certain conditions.

8.1 if-else Statement

int age = 18;
if (age >= 18) {
    System.out.println("You are an adult.");
} else {
    System.out.println("You are not an adult.");
}

8.2 Switch Statement

A switch statement can be used as an alternative to if-else when testing multiple conditions.

int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Unknown day");
}

9. Loops in Java

Loops are used to execute a block of code repeatedly. Java supports three types of loops: for, while, and do-while.

9.1 for Loop

A for loop is commonly used when you know how many times a block of code should run.

for (int i = 0; i < 5; i++) {
    System.out.println(i); // Outputs: 0, 1, 2, 3, 4
}

9.2 while Loop

The while loop continues to run as long as a condition is true.

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

9.3 do-while Loop

A do-while loop runs at least once, even if the condition is false.

int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

10. Functions in Java

Functions, also called methods, are reusable blocks of code designed to perform specific tasks. Here's how to create and call methods in Java:

10.1 Basic Function Example

public class Calculator {
    public static int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int result = add(10, 20);
        System.out.println("Result: " + result); // Outputs: 30
    }
}

10.2 Function Overloading

Java allows you to define multiple methods with the same name but different parameter lists, known as method overloading.

public class Calculator {
    public static int add(int a, int b) {
        return a + b;
    }

    public static double add(double a, double b) {
        return a + b;
    }

    public static void main(String[] args) {
        System.out.println(add(5, 10));      // Outputs: 15 (int version)
        System.out.println(add(5.5, 10.5));  // Outputs: 16.0 (double version)
    }
}

11. Type Casting in Java

Type casting is the process of converting one data type into another. Java supports two types of casting: implicit (automatic) and explicit (manual).

11.1 Implicit Casting

Java automatically converts smaller types to larger types (e.g., int to double).

int num = 100;
double converted = num; // Implicit casting (int to double)

11.2 Explicit Casting

Explicit casting is needed when converting larger types to smaller types (e.g., double to int).

double num = 9.78;
int converted = (int) num; // Explicit casting (double to int)
