1. History of C++

C++ was developed by Bjarne Stroustrup as an extension of the C programming language. It was first released in 1985 and has since become one of the most widely used programming languages.

Key features include:

  • Efficiency: C++ is known for its performance and efficiency, making it suitable for resource-intensive applications.
  • Object-Oriented: C++ supports object-oriented programming, which helps in organizing complex programs.

C++ has evolved over the years, with the latest standards adding many new features and improvements.

2. Key Features of C++

  • Performance: C++ provides high performance and is used in systems programming, game development, and real-time systems.
  • Flexibility: C++ supports multiple programming paradigms including procedural, object-oriented, and generic programming.
  • Rich Libraries: C++ has a vast standard library and numerous third-party libraries for various applications.
  • Community Support: C++ has a large and active community, providing extensive documentation and support.
  • Cross-Platform: C++ runs on various platforms, including Windows, macOS, and Linux.

3. Getting Started with C++

3.1 Setting Up Your Environment

To get started with C++, you need to install a compiler and set up an IDE. Follow these steps:

  • Download a Compiler: Install GCC (GNU Compiler Collection) for Windows, macOS, or Linux, or use an IDE like Visual Studio which includes a compiler.
  • Set Up an IDE: Use IDEs like Visual Studio, Code::Blocks, or CLion for an enhanced coding experience.

3.2 Writing Your First C++ Program

Once your environment is set up, create a new C++ file and write your first program:

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

3.3 Compiling and Running Your Program

To compile and run your program, use the command line or your IDE's build and run feature:

g++ hello.cpp -o hello
./hello

4. Variables and Data Types in C++

In C++, variables are used to store data, and each variable can hold different data types. C++ supports various data types, including:

4.1 Basic Data Types

  • int: Stores integers, e.g., int age = 30;
  • float: Stores floating-point numbers, e.g., float salary = 85000.50f;
  • double: Stores double-precision floating-point numbers, e.g., double pi = 3.14159;
  • char: Stores single characters, e.g., char grade = 'A';
  • bool: Stores true or false values, e.g., bool is_cpp_fun = true;

5. C++ Operators

Operators in C++ perform various operations on variables and values. Here are the types of operators:

5.1 Arithmetic Operators

Used for basic arithmetic operations:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ++ (Increment)
  • -- (Decrement)
int x = 5;
int y = 2;
std::cout << (x + y) << std::endl;  // Outputs: 7
std::cout << (x % y) << std::endl;  // Outputs: 1

5.2 Comparison Operators

Used to compare values:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
int a = 10;
int b = 20;
std::cout << (a == b) << std::endl;  // Outputs: 0 (false)
std::cout << (a < b) << std::endl;   // Outputs: 1 (true)

5.3 Logical Operators

Used to perform logical operations:

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)
bool is_adult = true;
bool has_id = false;
std::cout << (is_adult && has_id) << std::endl;  // Outputs: 0 (false)
std::cout << (is_adult || has_id) << std::endl;   // Outputs: 1 (true)
std::cout << (!is_adult) << std::endl;         // Outputs: 0 (false)

5.4 Assignment Operators

Used to assign values to variables:

  • = (Assigns a value)
  • += (Adds and assigns)
  • -= (Subtracts and assigns)
  • *= (Multiplies and assigns)
  • /= (Divides and assigns)
  • %= (Modulus and assigns)
int a = 5;
a += 3;  // a = a + 3, now a is 8
a *= 2;  // a = a * 2, now a is 16

6. Conditional Statements in C++

C++ supports several types of conditional statements that control the flow of execution based on certain conditions.

6.1 if-else Statement

int age = 18;
if (age >= 18) {
    std::cout << "You are an adult." << std::endl;
} else {
    std::cout << "You are not an adult." << std::endl;
}

6.2 else if Statement

The else if statement can be used to check multiple expressions:

int day = 3;
if (day == 1) {
    std::cout << "Monday" << std::endl;
} else if (day == 2) {
    std::cout << "Tuesday" << std::endl;
} else if (day == 3) {
    std::cout << "Wednesday" << std::endl;
} else {
    std::cout << "Unknown day" << std::endl;
}

7. Loops in C++

Loops are used to execute a block of code repeatedly. C++ supports for and while loops.

7.1 for Loop

A for loop is commonly used for iterating over a sequence (like a list or range):

for (int i = 0; i < 5; ++i) {
    std::cout << i << std::endl;  // Outputs: 0, 1, 2, 3, 4
}

7.2 while Loop

The while loop continues to run as long as a condition is true:

int i = 0;
while (i < 5) {
    std::cout << i << std::endl;
    ++i;
}

8. Functions in C++

Functions are reusable blocks of code designed to perform specific tasks. Here's how to create and call functions in C++:

8.1 Basic Function Example

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(10, 20);
    std::cout << "Result: " << result << std::endl;  // Outputs: 30
    return 0;
}

8.2 Function with Default Parameters

You can define functions with default parameter values:

void greet(std::string name = "World") {
    std::cout << "Hello, " << name << std::endl;
}

int main() {
    greet();          // Outputs: Hello, World
    greet("Alice");  // Outputs: Hello, Alice
    return 0;
}
