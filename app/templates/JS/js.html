1. History of JavaScript

JavaScript was developed by Brendan Eich in 1995 while working for Netscape Communications Corporation. Initially called Mocha, then LiveScript, it was finally renamed JavaScript.

Key features include:

  • Client-side Scripting: JavaScript is primarily used for creating interactive web pages.
  • Lightweight and Fast: JavaScript is a lightweight programming language that executes quickly in the browser.

JavaScript has evolved significantly over the years, with ECMAScript standards ensuring its continuous development and improvement.

2. Key Features of JavaScript

  • Interactivity: JavaScript adds interactivity to web pages, allowing dynamic content updates.
  • Event-driven: JavaScript can handle and react to user events like clicks and keypresses.
  • Platform Independence: JavaScript runs in any web browser, making it platform-independent.
  • Versatile: JavaScript can be used for both client-side and server-side development (e.g., using Node.js).
  • Rich Libraries: JavaScript has a vast ecosystem of libraries and frameworks (e.g., React, Angular, Vue.js).

3. Getting Started with JavaScript

3.1 Setting Up Your Environment

To get started with JavaScript, you need a web browser and a text editor. Follow these steps:

  • Choose a Text Editor: Use editors like Visual Studio Code, Sublime Text, or Atom for an enhanced coding experience.
  • Open Your Browser: Modern browsers like Chrome, Firefox, or Edge have built-in JavaScript engines.

3.2 Writing Your First JavaScript Program

Once your environment is set up, create a new HTML file and write your first JavaScript program:

    Hello, World!

3.3 Running Your Program

  • Open Your HTML File: Open the HTML file in your web browser. You will see "Hello, World!" on the web page.
  • View Console Output: Open the browser's Developer Tools (usually F12 or right-click and select "Inspect") and go to the "Console" tab to see the "Hello, World!" message.

4. Variables and Data Types in JavaScript

In JavaScript, variables are used to store data, and each variable can hold different data types. JavaScript supports various data types, including:

4.1 Basic Data Types

  • Number: Stores numbers, e.g., let age = 30;
  • String: Stores text, e.g., let name = "John";
  • Boolean: Stores true or false values, e.g., let isJavaScriptFun = true;
  • Undefined: A variable that has been declared but not assigned a value, e.g., let x;
  • Null: Represents the intentional absence of any value, e.g., let y = null;

5. JavaScript Operators

Operators in JavaScript perform various operations on variables and values. Here are the types of operators:

5.1 Arithmetic Operators

Used for basic arithmetic operations:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ++ (Increment)
  • -- (Decrement)
let x = 5;
let y = 2;
console.log(x + y);  // Outputs: 7
console.log(x % y);  // Outputs: 1

5.2 Comparison Operators

Used to compare values:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
let a = 10;
let b = 20;
console.log(a == b);  // Outputs: false
console.log(a < b);   // Outputs: true

5.3 Logical Operators

Used to perform logical operations:

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)
let isAdult = true;
let hasID = false;
console.log(isAdult && hasID);  // Outputs: false
console.log(isAdult || hasID);  // Outputs: true
console.log(!isAdult);          // Outputs: false

5.4 Assignment Operators

Used to assign values to variables:

  • = (Assigns a value)
  • += (Adds and assigns)
  • -= (Subtracts and assigns)
  • *= (Multiplies and assigns)
  • /= (Divides and assigns)
  • %= (Modulus and assigns)
let a = 5;
a += 3;  // a = a + 3, now a is 8
a *= 2;  // a = a * 2, now a is 16

6. Conditional Statements in JavaScript

JavaScript supports several types of conditional statements that control the flow of execution based on certain conditions.

6.1 if-else Statement

let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are not an adult.");
}

6.2 else if Statement

The else if statement can be used to check multiple expressions:

let day = 3;
if (day == 1) {
    console.log("Monday");
} else if (day == 2) {
    console.log("Tuesday");
} else if (day == 3) {
    console.log("Wednesday");
} else {
    console.log("Unknown day");
}

7. Loops in JavaScript

Loops are used to execute a block of code repeatedly. JavaScript supports for and while loops.

7.1 for Loop

A for loop is commonly used for iterating over a sequence (like a list or range):

for (let i = 0; i < 5; i++) {
    console.log(i);  // Outputs: 0, 1, 2, 3, 4
}

7.2 while Loop

The while loop continues to run as long as a condition is true:

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

8. Functions in JavaScript

Functions are reusable blocks of code designed to perform specific tasks. Here's how to create and call functions in JavaScript:

8.1 Basic Function Example

function add(a, b) {
    return a + b;
}

let result = add(10, 20);
console.log("Result: " + result);  // Outputs: 30

8.2 Function with Default Parameters

You can define functions with default parameter values:

function greet(name = "World") {
    console.log("Hello, " + name);
}

greet();          // Outputs: Hello, World
greet("Alice");  // Outputs: Hello, Alice
