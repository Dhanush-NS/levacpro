1. History of C

The C programming language was developed by Dennis Ritchie at Bell Labs in the early 1970s. It was designed to be a system programming language and has since become one of the most widely used programming languages.

Key features include:

  • Efficiency: C is known for its performance and efficiency, making it suitable for system-level programming.
  • Low-Level Access: C provides low-level access to memory and allows for direct manipulation of hardware.

C has influenced many other programming languages, and its syntax and concepts are still widely used today.

2. Key Features of C

  • Portability: C programs can be easily ported to different platforms with minimal changes.
  • Powerful: C provides powerful constructs and low-level access, making it suitable for system programming, embedded systems, and more.
  • Rich Libraries: C has a vast standard library that provides functions for various tasks.
  • Community Support: C has a large and active community, providing extensive documentation and support.
  • Foundation Language: Many modern programming languages are based on or influenced by C.

3. Getting Started with C

3.1 Setting Up Your Environment

To get started with C, you need to install a compiler and set up an IDE. Follow these steps:

  • Download a Compiler: Install GCC (GNU Compiler Collection) for Windows, macOS, or Linux, or use an IDE like Visual Studio which includes a compiler.
  • Set Up an IDE: Use IDEs like Visual Studio, Code::Blocks, or CLion for an enhanced coding experience.

3.2 Writing Your First C Program

Once your environment is set up, create a new C file and write your first program:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

3.3 Compiling and Running Your Program

To compile and run your program, use the command line or your IDE's build and run feature:

gcc hello.c -o hello
./hello

4. Variables and Data Types in C

In C, variables are used to store data, and each variable can hold different data types. C supports various data types, including:

4.1 Basic Data Types

  • int: Stores integers, e.g., int age = 30;
  • float: Stores floating-point numbers, e.g., float salary = 85000.50f;
  • double: Stores double-precision floating-point numbers, e.g., double pi = 3.14159;
  • char: Stores single characters, e.g., char grade = 'A';

5. C Operators

Operators in C perform various operations on variables and values. Here are the types of operators:

5.1 Arithmetic Operators

Used for basic arithmetic operations:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • ++ (Increment)
  • -- (Decrement)
int x = 5;
int y = 2;
printf("%d\n", x + y);  // Outputs: 7
printf("%d\n", x % y);  // Outputs: 1

5.2 Comparison Operators

Used to compare values:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
int a = 10;
int b = 20;
printf("%d\n", a == b);  // Outputs: 0 (false)
printf("%d\n", a < b);   // Outputs: 1 (true)

5.3 Logical Operators

Used to perform logical operations:

  • && (Logical AND)
  • || (Logical OR)
  • ! (Logical NOT)
int is_adult = 1;
int has_id = 0;
printf("%d\n", is_adult && has_id);  // Outputs: 0 (false)
printf("%d\n", is_adult || has_id);   // Outputs: 1 (true)
printf("%d\n", !is_adult);         // Outputs: 0 (false)

5.4 Assignment Operators

Used to assign values to variables:

  • = (Assigns a value)
  • += (Adds and assigns)
  • -= (Subtracts and assigns)
  • *= (Multiplies and assigns)
  • /= (Divides and assigns)
  • %= (Modulus and assigns)
int a = 5;
a += 3;  // a = a + 3, now a is 8
a *= 2;  // a = a * 2, now a is 16

6. Conditional Statements in C

C supports several types of conditional statements that control the flow of execution based on certain conditions.

6.1 if-else Statement

int age = 18;
if (age >= 18) {
    printf("You are an adult.\n");
} else {
    printf("You are not an adult.\n");
}

6.2 else if Statement

The else if statement can be used to check multiple expressions:

int day = 3;
if (day == 1) {
    printf("Monday\n");
} else if (day == 2) {
    printf("Tuesday\n");
} else if (day == 3) {
    printf("Wednesday\n");
} else {
    printf("Unknown day\n");
}

7. Loops in C

Loops are used to execute a block of code repeatedly. C supports for and while loops.

7.1 for Loop

A for loop is commonly used for iterating over a sequence (like a list or range):

for (int i = 0; i < 5; ++i) {
    printf("%d\n", i);  // Outputs: 0, 1, 2, 3, 4
}

7.2 while Loop

The while loop continues to run as long as a condition is true:

int i = 0;
while (i < 5) {
    printf("%d\n", i);
    ++i;
}

8. Functions in C

Functions are reusable blocks of code designed to perform specific tasks. Here's how to create and call functions in C:

8.1 Basic Function Example

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(10, 20);
    printf("Result: %d\n", result);  // Outputs: 30
    return 0;
}

8.2 Function with Default Parameters

Note: C does not support default parameters natively like C++.
